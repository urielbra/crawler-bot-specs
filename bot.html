About GZUBot

Learn about GZUBot, the web crawler for CEFET RI Studing.

Developed by: Guilherme G. Gomes & Uriel B. Braga

GZUBot is the web crawler for CEFET RI Studing. No Products use GZUBot.

Verifying GZUBot user agent

The user-agent string contains ”GZUBot” and other information. This is the format: Mozilla/5.0 (Device; OS_version) AppleWebKit/WebKit_version (KHTML, like Gecko) Version/Safari_version Safari/WebKit_version (GZUBot (https://urielbra.github.io/bot.html))

Customizing robot.txt rules

GZUBot respects standard robots.txt directives that are targeted at GZUBot. User-agent: * # any robot
Disallow: /not-allowed/ # disallow this directory

Crawling dates

This bot has been active in periods between July/02/2021 and July/23/2021.

Crawling purposes

This bot was developed by CEFET-MG students in Brazil for study purposes only, so it has no commercial or business intention.
